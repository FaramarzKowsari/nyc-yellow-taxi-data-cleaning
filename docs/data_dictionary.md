# Data dictionary

The authoritative field definitions are maintained by the NYC Taxi & Limousine Commission (TLC):

- Yellow Taxi Trip Records Data Dictionary: https://www.nyc.gov/assets/tlc/downloads/pdf/data_dictionary_trip_records_yellow.pdf
- Trip Record Data portal: https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

The project preserves source column names until the canonicalization stage. Derived columns are documented in `docs/methodology.md`.
