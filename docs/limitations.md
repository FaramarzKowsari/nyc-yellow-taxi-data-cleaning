# Limitations

1. TLC states that trip records are submitted by technology providers and are not guaranteed to be perfectly accurate or complete.
2. A suspicious record is not automatically false. The pipeline separates hard-invalid rules from soft anomaly flags.
3. Outlier thresholds are operational choices for data quality analysis, not universal truths.
4. Fare-component reconciliation can be affected by evolving surcharges and fields. It is therefore a flag, not a deletion rule.
5. The repository does not infer passenger identity or driver behavior.
